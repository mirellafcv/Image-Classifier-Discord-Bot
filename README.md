## Image Classifier Discord Bot

# Overview
*This project is a Discord bot used for animal identification. It helps people to classify animals with the help of percentages that are given after bot's analysis.*

# Features
*The main feature of this project is analyze a given image containing animals and classify it based on different species, including percentages of certainty.*

# Code for using the bot
*This bot is activated through the use of the command "?analisar_foto" in your Discord server.*
